Product Demo REST API Service

GET /products to see all the products.
GET /products?category=dog to see all of the products in the 'dog' category.
GET /products/{id} retrieves a product from the DB using an ID.
PUT /products/{id} updates a product using an ID.
DELETE /products/{id} removes a product from the DB using an ID.
POST /products creates an new product.

Example JSON Data:

{"name": "Bowser", "category": "Dog", "gender": "MALE", "available": "True", "birthday": "2022-05-11"}
